Generuje plik candl-report.html otwierany w przeglądarce. Interaktywne filtrowanie, sortowanie, grupowanie. Shareable — można wysłać teamowi. Działa offline.

C — Hybrid (terminal summary + HTML/JSON export)

Terminal pokazuje zwięzłe podsumowanie z progress barami. Na końcu pyta: "Otwórz raport HTML?" / "Zapisz JSON?". HTML raport i JSON to opcjonalne wyjście — terminal pozostaje głównym UX.
